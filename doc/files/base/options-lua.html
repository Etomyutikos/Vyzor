Properties

Option Properties

DrawOrderTODO: Will determine z-layer ordering for Border Frames.
BordersReturns a table containing options.  Sets Border options via a table.
ConsoleHeightSets and gets a user-defined main console height.
HandleBordersGets and sets a value that determines whether or not Vyzor handles Mudlet’s borders.  If true, then Vyzor will always resize Mudlet’s borders.  If auto, it will only assume control after the Vyzor.HUD:Draw() has been called.
